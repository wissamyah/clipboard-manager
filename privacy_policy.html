üîê Data Security

How Your Data is Protected

  • Chrome Storage API: Uses Chrome's secure storage system with sandboxing
  • Local-Only Access: Data is only accessible through the extension interface
  • No Network Requests: The extension doesn't make any network calls for your data
  • Automatic Cleanup: Old items (90+ days) are automatically removed to manage storage
  • User Control: You have full control to delete any or all data at any time

Data Compression & Optimization

To maximize storage efficiency within Chrome's 5MB limit:

  • Text data is compressed to save space
  • Duplicate entries are automatically detected and removed
  • Unused tags are cleaned up periodically
  • Manual optimization available via the Settings panel

üë§ User Rights & Control

You Have Complete Control

  • Access: View all stored data through the extension interface
  • Export: Export your entire clipboard history at any time
  • Delete: Remove individual items or clear all history
  • Optimize: Use the optimization feature to clean up old data
  • Uninstall: Removing the extension deletes all associated data

Data Retention

  • Data is kept until you manually delete it or uninstall the extension
  • Automatic cleanup removes items older than 90 days (configurable)
  • Favorites and pinned items are protected from automatic cleanup
  • No backups are kept after deletion
